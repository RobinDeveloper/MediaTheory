My machine learning journey

December 21, 2017, Robin Opheij

When I started this year I had a clear goal in mind.
Get myself acquainted with machine learning and genetic algorithms. I already had done some small stuff with machine learning in processing and java. But I really wanted to step it up and create a scene of agents that flock together (in space) and potentially even have full on space battles.
The last part might be a bit over the top for my first algorithm but I should be able to get atleast the flocking done.

I wanted to set a few rules for the boids that they should know overtime.

  • Don't fly into eachother.
  • Stay close to eachother.
  • Folow a target or the first boid of the flock.
    first boid should know to just fly anywhere it wants.

So where do I start? because there is no way my first neural network project can be 3D flocking boids.
I started to look around and ended up with Text color on solid color test. I made a quick node diagram and the calculations I needed to make the neural network work.
That node diagram looks something like this but with RGB as inputs and Black Or White as output. The one I am going to show was the first pass for the flocking behaviour. I am now working on a new and improved version of this diagram.

I quickly got stuck though and started working on a new Idea.
a genetic algorithm without hidden layers. I now started making a cameleon simulator.
every agent will overtime learn to adapt to it's environment by checking the magnitude between their color and the environment by then making children who pick random genes from it parents. the only parents that are allowed to "breed" are the top 50% of the agents from the last generation.
the top 50% gets decided form what their fitness is which I calculate trough the magnitude as I said above.

Once I got this figured out I tried using this system for the boids and by giving them a move to position to a target but I decided to drop it to dive deeper into writing a real real neural network where I can then put some variables in as input and use the output in the the algorithm. 1 in this neural network I use the feed forward technique:

In the end I realized how easy the part of writing the neural network actually is and that the real hard part is writing the algorithm which you train upon. I am aware that I only so far have written Deep Neural Networks with genetic algorithm and that those are a lot easier to write and understand than for instance deep reinforcement learning and I am working on some small c# projects where I experiment with expected results and teaching the neural networks that way.
But back to the 2d flocking. In this project I had a few boids that didn't care about each other and only wanted to fly towards a target by angling them self the right way so they can move towards the target. and now I'm working on this making this work in 3D with all the rules I set above.

Whilst I was working on the flocking one of the other kids on my school showed of his genetic car driving. I saw this and completely forgot that pretty much everyone interested in machine learning in a 3D sense made this genetic algorithm and I went and tried making one. this ended up being way easier than I thought I spend around 2 days on this little project. I did write a new neural network for this one because I thought I could make it more efficient. because before I just ran the input trough the neural network and didn't really set up any biases but now I had the neural network and let all the calculation be done by the genetic manager. this way I could also swap it out for a reinforcement learning model or any other form of deep learning.
So in the next gif you see some of the training. the green parts are checkpoints which they have to go trough to have their new target set and adjust their nodes. When one reaches the last checkpoint the training is done and they have done it. I call this a genetic car driving experiment but these cars might aswell be rats trying to find their way trough the maze. but fot this example lets just say they are cars trying to maneuver them self trough some weird london streets.

So it's been a while since I updated this post but I have come a far way experimenting with new neural networks and algorithms. I will quickly show you the first neural network I wrote and then move on to how I make my neural nets now. You can also read my blog post about how to write a neural network and then use it for a programm in unity c# if you don't even know how and where to start like me in the beginning. luckly this all isn't too hard.
